# Mathematical Operation Web Calculator

##  Description

This is a simple web-based calculator that performs basic mathematical operations using **JavaScript**.
The application allows users to input two numbers and perform operations such as addition, subtraction, multiplication, division, and power.

This project demonstrates basic **JavaScript functions, DOM manipulation, and form input handling**.

---

##  Features

* Add two numbers
* Subtract two numbers
* Multiply two numbers
* Divide two numbers
* Power operation
* Simple HTML form interface

---

##  Technologies Used

* HTML5
* JavaScript
* Basic DOM Manipulation

##  How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/math-operation-calculator.git
```

2. Open the project folder.

3. Run the project by opening the **index.html** file in your browser.

---

##  How It Works

1. Enter the **first number**.
2. Enter the **second number**.
3. Click the operation button:

   * Add
   * Subtract
   * Multiply
   * Divide
   * Power
4. The result will automatically appear in the **Answer field**.
